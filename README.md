# SMART-AI: LVO-AngioNet System

The **LVO-AngioNet** system addresses two fundamental challenges in large-vessel occlusion (LVO) imaging:

- **Sparse expert annotations**
- **Highly heterogeneous thrombus appearance**

It achieves robust performance through a synergistic integration of **synthetic data generation** and **anatomically constrained reconstruction**.
